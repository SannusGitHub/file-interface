# About
This repository contains the project code for file-interface, a simple hostable front-end and back-end service for file management.

The project was initially designed as a personal project so I can better navigate and manage through my file storage server.

## Local Installation
1. Download the repository source code from this repository.
2. Run with `go run main.go`

## Features
* Front-end
     * Basic interface
        * Managing files (Upload, Download, Deletion)
        * Folder navigation
        * Setting scale (XS / S / M / L)
        * Preview modal & file info
* Back-end
     * Upload, download, file list & meta end-points
